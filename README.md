# Nickolas Portfolio

Portfólio em Next.js focado em sistemas embarcados, firmware, IoT, telemetria e integração de dados.

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse:

```text
http://localhost:3000
```

## Build

```bash
npm run build
npm run start
```

## Deploy

### Netlify

- Framework preset: Next.js
- Install command: `npm install`
- Build command: `npm run build`

### Vercel

Detecta Next.js automaticamente.
